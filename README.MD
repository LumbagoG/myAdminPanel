# Админ панель на react 

## О приложении
Это приложение админ панели на react

## Доступные скрипты

### Запуск приложения в режиме разработки

```bash
    npm run start
```

### Запуск сборки приложения

```bash
    npm run build
```

### Запуск собранной версии приложения

```bash
    npm run build:start
```

### Запуск анализатора бандла

```bash
    npm run analyze
```
